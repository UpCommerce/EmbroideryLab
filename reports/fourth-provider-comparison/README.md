# Fourth provider comparison

This package contains 19 source images from the three-provider historical report.

## Workflow

1. Use every image in `sources/` as input for the fourth provider.
2. Save each generated preview in `results-fourth-provider/`.
3. Keep the exact source ID at the beginning of the result filename.
4. Do not overwrite or rename the source images.

Example:

- Source: `C3-001-S01__Arcon-1.png`
- Result: `C3-001-S01__fourth-provider.png`

The text after the source ID may be changed. PNG, JPG, JPEG and WebP results are accepted.
